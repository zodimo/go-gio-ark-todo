# go-gio-ark-todo

A play project for learning and experimenting with ARK ECS (Entity Component System) and gioui.

## Overview

This project demonstrates how to build interactive applications using:
- **ARK ECS**: A fast, type-safe Entity Component System for Go
- **gioui v0.9.0**: Immediate mode GUI library for Go

## Prerequisites

- Go 1.24.3 or later
